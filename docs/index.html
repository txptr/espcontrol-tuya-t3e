🎛️ EspControl Tuya T3E

Smart Home Control Panel

📦 What This Is

This is the web server for the Tuya T3E ESP32-S3 smart home control panel running EspControl.

🚀 Getting Started

If you're seeing this page, your device is:

  • ✅ Connected to WiFi
  • ✅ Running EspControl firmware
  • ✅ Ready to configure

⚙️ Device URL

Access your device's control panel at:

http://<device-ip>/

Replace <device-ip> with your device's IP address from Home Assistant or your router.
